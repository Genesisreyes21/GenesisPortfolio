# Genesis' Portfolio

A personal portfolio showcasing interactive projects, creative technology work, and selected pieces. Built as a single-page experience with animated visuals and a project modal.

## Features
- Full-screen landing with animated background
- Work grid with project previews
- Modal project details with links
- About section with profile image and contact button
- Responsive layout for desktop and mobile

## Tech Stack
- HTML, CSS, JavaScript
- Three.js (background scene)
- GSAP (page and modal animations)

## Getting Started
1. Open `index.html` in a browser.
2. If you are using a local server, serve the folder and open the provided URL.

## Customization
- Update project data in your main JS file (look for the `projects` array).
- Replace images in the project image paths used by the `projects` array.
- Edit styles in your main CSS file.
- Update contact button behavior to open your email or Gmail compose link.

## Project Structure
- `index.html` - main page
- `styles.css` - styling
- `script.js` - interactions and animations
- `*.png` - project images

## Deployment
You can deploy this portfolio using any static hosting:
